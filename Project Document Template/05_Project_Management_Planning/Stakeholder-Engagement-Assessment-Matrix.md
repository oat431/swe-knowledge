## 1. Purpose

> This matrix tracks current vs. desired stakeholder engagement levels, identifies gaps, and defines actions to close those gaps.

## 2. Engagement Levels

| Level | Code | Description |
|-------|------|-------------|
| **Unaware** | U | [Does not know about the project or its impacts] |
| **Resistant** | R | [Knows but is opposed to the change] |
| **Neutral** | N | [Knows but is neither supportive nor resistant] |
| **Supportive** | S | [Knows and supports the change] |
| **Leading** | L | [Actively engaged in driving the change forward] |
