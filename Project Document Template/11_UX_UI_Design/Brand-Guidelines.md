## 1. Purpose

> Brand guidelines ensure consistent visual identity across all touchpoints — logo, colors, typography, and voice.

## 2. Logo

### 2.1 Logo Variants

| Variant | Usage | Background |
|---------|-------|-----------|
| [Primary — Full Color] | [Default usage] | [White or light backgrounds] |
| [Monochrome — Dark] | [Print, single-color] | [White or light backgrounds] |
| [Monochrome — Light] | [Dark backgrounds] | [Dark or colored backgrounds] |
| [Icon Only] | [Favicon, app icon, small spaces] | [Any] |

### 2.2 Logo Rules

| Rule | Specification |
|------|--------------|
| [Minimum Size] | [32px height (digital), 12mm (print)] |
| [Clear Space] | [1x logo height on all sides] |
| [Background] | [Never place on busy/patterned backgrounds] |
| [Modification] | [Never stretch, rotate, recolor, or add effects] |
| [Co-branding] | [Partner logo same size, separated by vertical line] |

### 2.3 Logo Don'ts

| ❌ Don't | Why |
|---------|-----|
| [Stretch or distort] | [Breaks brand recognition] |
| [Change colors] | [Inconsistent identity] |
| [Place on busy backgrounds] | [Reduces legibility] |
| [Add shadows or effects] | [Unprofessional] |
| [Use below minimum size] | [Unreadable] |

## 5. Brand Voice

| Attribute | Description | Example |
|----------|-------------|---------|
| [Professional] | [Knowledgeable, trustworthy] | ["Your request has been reviewed"] |
| [Clear] | [Simple, direct, jargon-free] | ["Submit your request" not "Initiate workflow"] |
| [Helpful] | [Supportive, guiding] | ["Here's what to do next"] |
| [Confident] | [Assured, not hesitant] | ["Request approved" not "Request might be approved"] |

## 6. Brand Tone by Context

| Context | Tone | Example |
|---------|------|---------|
| [Success messages] | [Positive, encouraging] | ["Great! Your request has been submitted."] |
| [Error messages] | [Helpful, not blaming] | ["Something went wrong. Please try again."] |
| [Empty states] | [Guiding, helpful] | ["No requests yet. Create your first one!"] |
| [Notifications] | [Informative, concise] | ["Your request #123 has been approved."] |
