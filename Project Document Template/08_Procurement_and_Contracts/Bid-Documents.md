## 1. Purpose

> This document provides templates for Request for Information (RFI), Request for Proposal (RFP), and Request for Quotation (RFQ).

## 2. Document Type Selection

| Type | When to Use | Purpose | Response Detail |
|------|------------|---------|----------------|
| **RFI** | [Market research, early exploration] | [Gather information about vendor capabilities] | [Qualitative — no pricing] |
| **RFP** | [Complex services, need solutions] | [Get proposed solutions and pricing] | [Technical + pricing + approach] |
| **RFQ** | [Well-defined scope, price-focused] | [Get competitive pricing for known requirements] | [Pricing + delivery timeline] |
