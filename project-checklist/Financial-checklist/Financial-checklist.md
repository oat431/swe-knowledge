## Phase 1: Expense Tracking (Month 1-2)

### Week 1: Setup
- [ ] Create Excel expense tracker with columns: Date, Category, Description, Amount, Running Total
- [ ] Add category dropdown: Fixed / Variable / Discretionary
- [ ] Add summary formulas: `=SUMIF(B:B,"Fixed",D:D)` for each category
- [ ] Add remaining budget formula: `=32000-SUM(D:D)`
- [ ] Add savings rate formula: `=(32000-SUM(D:D))/32000`
- [ ] Set conditional formatting: Green (positive), Red (negative)
- [ ] Set daily phone reminder: "Did you track expenses today?"
- [ ] Track Day 1 expenses

### Week 2-4: Build the Habit
- [ ] Track every expense immediately (not later)
- [ ] Categorize each expense correctly
- [ ] Complete Week 1 — review total spending
- [ ] Complete Week 2 — identify any patterns
- [ ] Complete Week 3 — note any leaks (forgotten subscriptions)
- [ ] Complete Week 4 — calculate first month total

### Week 5-8: Observe & Analyze
- [ ] Track consistently for 30 days (no judgment, just observe)
- [ ] Calculate actual monthly surplus (Income - Expenses)
- [ ] Identify top 3 spending categories
- [ ] Identify biggest "leak" (unexpected high spend)
- [ ] Calculate average daily spending
- [ ] Compare Week 1 vs Week 4 spending patterns
- [ ] Complete 60-day tracking period
- [ ] Finalize actual surplus number

### Phase 1 Deliverables
- [ ] 60 days of complete expense data
- [ ] Actual monthly surplus calculated: __________ THB
- [ ] Top 3 expense categories identified
- [ ] Spending patterns documented
- [ ] Ready for Phase 2

## Phase 3: Basic Investing (Month 8+)

### Step 1: Tax-Advantaged Accounts (Priority)
- [ ] Research RMF (Retirement Mutual Fund) options
- [ ] Research SSF (Super Savings Fund) options
- [ ] Calculate max tax deduction: 30% of income = 9,600 THB/month
- [ ] Open RMF account (recommended: 5,000 THB/month)
- [ ] Open SSF account (recommended: 4,600 THB/month)
- [ ] Set up automatic monthly investment

### Step 2: SET Index Fund (Diversified Thai Equity)
- [ ] Research SET50 index funds
- [ ] Compare fund performance and fees
- [ ] Open brokerage account (if needed)
- [ ] Start DCA with 2,000-3,000 THB/month
- [ ] Track performance quarterly

### Step 3: Individual Stocks (After 1 Year of Index Investing)
- [ ] Study fundamental analysis basics
- [ ] Research 5-10 Thai blue-chip stocks
- [ ] Start with small position (5,000 THB)
- [ ] Diversify across sectors
- [ ] Never invest more than 5% in single stock

### Step 4: Crypto (Optional — Max 5-10% of Portfolio)
- [ ] Research Bitcoin and Ethereum basics
- [ ] Choose Thai exchange (Bitkub, Satang Pro)
- [ ] Start with small amount (1,000-2,000 THB)
- [ ] Use DCA strategy
- [ ] Set stop-loss rules
- [ ] Never invest more than you can afford to lose

### Investment Rules
- [ ] Never invest emergency fund
- [ ] Diversify across asset classes
- [ ] Use DCA (Dollar-Cost Averaging)
- [ ] Review portfolio quarterly
- [ ] Rebalance annually
- [ ] Stay invested long-term (5+ years)

### Phase 3 Deliverables
- [ ] RMF/SSF accounts open and funded
- [ ] SET index fund investment active
- [ ] Investment allocation plan documented
- [ ] Tax benefits optimized
