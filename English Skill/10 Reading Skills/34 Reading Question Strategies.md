# Reading Question Strategies

Knowing the question types helps you find answers faster. Each type has a specific strategy — treat them differently.

---

## IELTS Reading Question Types

### 1. True / False / Not Given

The most common (and most confusing) type.

| Answer | Meaning | How to Decide |
|--------|---------|---------------|
| **True** | The passage agrees with the statement | Find evidence that confirms it |
| **False** | The passage contradicts the statement | Find evidence that disagrees |
| **Not Given** | The passage doesn't mention it at all | No evidence either way |

> **The trap:** "Not Given" feels wrong because students WANT to find an answer. If the passage doesn't say anything about it, it's Not Given — even if you think it's probably true.

### Example
> **Passage:** "The university was founded in 1850 and has 15,000 students."

| Statement | Answer | Why |
|-----------|--------|-----|
| The university was established in 1850. | **True** | "founded" = "established" |
| The university has 20,000 students. | **False** | Passage says 15,000 |
| The university is located in London. | **Not Given** | Passage doesn't mention location |

### 2. Matching Headings

Match each paragraph to a summary heading.

| Strategy | Steps |
|----------|-------|
| 1 | Read all headings first |
| 2 | Skim each paragraph — focus on first and last sentences |
| 3 | Match the **main idea**, not a detail |
| 4 | Eliminate used headings |

### 3. Matching Information

"Which paragraph contains the following information?"

| Strategy | Steps |
|----------|-------|
| 1 | Read the statements |
| 2 | Identify keywords in each statement |
| 3 | Scan the passage for those keywords or synonyms |
| 4 | Confirm the paragraph actually contains the information |

### 4. Sentence Completion

Complete a sentence with words from the passage.

| Rule | Detail |
|------|--------|
| **Word limit** | Usually "NO MORE THAN TWO/THREE WORDS" — obey this strictly |
| **Exact words** | Use words directly from the passage |
| **Grammar** | The completed sentence must be grammatically correct |
| **Spelling** | Must be spelled correctly |

### 5. Summary Completion

Fill in gaps in a summary of the passage.

| Strategy | Steps |
|----------|-------|
| 1 | Read the summary to understand the flow |
| 2 | Locate the relevant section in the passage |
| 3 | Find the exact word(s) that fit both meaning and grammar |
| 4 | Check the word limit |

### 6. Multiple Choice

Choose the correct answer from A, B, C, D.

| Strategy | Steps |
|----------|-------|
| 1 | Read the question — don't read the options yet |
| 2 | Find the relevant section in the passage |
| 3 | Form your own answer first |
| 4 | Match it to the options |
| 5 | Eliminate wrong answers |

## TOEFL Reading Question Types

### 1. Factual Information

"According to paragraph 1, which of the following is true about X?"

| Strategy | Steps |
|----------|-------|
| 1 | Identify keywords in the question |
| 2 | Scan the paragraph for those keywords |
| 3 | Read the sentence and surrounding context |
| 4 | Match to the correct option |

### 2. Negative Factual Information

"Which of the following is NOT mentioned in the passage?"

| Strategy | Steps |
|----------|-------|
| 1 | Check each option against the passage |
| 2 | Three options WILL be mentioned |
| 3 | The one that ISN'T mentioned is your answer |
| 4 | Don't confuse "not mentioned" with "not true" |

### 3. Inference

"What can be inferred from paragraph 2 about X?"

See [[33 Inference & Implied Meaning]] for full strategy.

### 4. Rhetorical Purpose

"Why does the author mention X?"

| Purpose | Answer Pattern |
|---------|---------------|
| To provide an example | "to illustrate..." |
| To support an argument | "to demonstrate..." |
| To compare | "to contrast..." |
| To explain a term | "to define..." |
| To challenge a view | "to question..." |

### 5. Vocabulary in Context

"The word X in the passage is closest in meaning to..."

| Strategy | Steps |
|----------|-------|
| 1 | Re-read the sentence with the word |
| 2 | Cover the word — what would fit? |
| 3 | Check each option in the sentence |
| 4 | The one that makes sense in context is correct |

> Don't just pick the definition you know. The word might be used differently in context.

### 6. Sentence Simplification

"Which of the following best expresses the essential information in the highlighted sentence?"

| Rule | Detail |
|------|--------|
| Must keep | The main idea |
| Can remove | Extra details, examples, repetition |
| Must not | Contradict the original or add new information |

### 7. Insert Text

"Where would the following sentence best fit?"

| Strategy | Steps |
|----------|-------|
| 1 | Read the sentence to insert — look for linking words |
| 2 | "This," "these," "such" → must refer to something BEFORE |
| 3 | "However," "therefore" → must connect to what's BEFORE and AFTER |
| 4 | Test each position — does it flow logically? |

### 8. Prose Summary

Select 3 sentences that summarize the passage.

| Strategy | Steps |
|----------|-------|
| 1 | Eliminate options that are too specific (details, not main ideas) |
| 2 | Eliminate options that are inaccurate |
| 3 | Choose the 3 that together capture the passage's main ideas |

## General Time Management

| IELTS | TOEFL |
|-------|-------|
| 3 passages, 60 minutes | 2-3 passages, 54-72 minutes |
| ~20 minutes per passage | ~18 minutes per passage |
| Spend 2 min skimming, then answer | Same approach |
| Mark difficult questions, come back | Same — don't get stuck |

## ⚠️ Thai Speaker Traps

| Trap | Problem | Fix |
|------|---------|-----|
| Overthinking "Not Given" | "It could be true, so maybe it's True?" | If the passage doesn't say it, it's Not Given |
| Ignoring word limits | Writing 4 words when limit is 2 | Check the instruction EVERY time |
| Choosing the first "good" answer | Missing a better option | Read ALL options before choosing |
| Not returning to the passage | Answering from memory | ALWAYS re-read the relevant text |
| Translating questions to Thai | Losing time and accuracy | Practice reading questions in English directly |
