# Note-Taking for Listening

IELTS and TOEFL listening sections play audio **only once**. You can't rewind. Effective note-taking is the difference between remembering and guessing.

---

## Why Note-Taking Matters

| Test | Format | Challenge |
|------|--------|-----------|
| **IELTS** | 4 sections, 30 minutes, audio plays once | Must answer while listening |
| **TOEFL** | 3-4 lectures + 2-3 conversations, 41-57 minutes | Long lectures (5 min) — can't remember everything |

---

## The Cornell Note-Taking System (Adapted for Tests)

```
┌─────────────────────────────────────────────────┐
│  KEYWORDS / QUESTIONS    │  NOTES / DETAILS      │
│  (write BEFORE or DURING)│  (write DURING)        │
│                          │                        │
│  topic?                  │  environment           │
│  when?                   │  since 2010            │
│  problem?                │  pollution increasing   │
│  solution?               │  new policy proposed    │
│                          │                        │
├──────────────────────────┴────────────────────────┤
│  SUMMARY (write AFTER)                            │
│  New policy to reduce pollution since 2010         │
└──────────────────────────────────────────────────┘
```

## What to Write Down

### ✅ DO Note These

| Type | Why | Example |
|------|-----|---------|
| **Numbers & dates** | Hard to remember exactly | "2019," "50%," "3 million" |
| **Names** | Often tested | "Dr. Smith," "Oxford University" |
| **Key nouns** | The topic and subtopics | "pollution," "policy," "research" |
| **Signal words** | Tell you what's important | "The key point is...," "What's crucial..." |
| **Contrasts** | "but," "however," "on the other hand" | Often tested |
| **Examples** | Support main points | "for instance," "such as" |
| **Cause-effect** | "because," "therefore," "as a result" | Logical connections |
| **Sequence** | "first," "then," "finally" | Process or steps |

### ❌ DON'T Waste Time On

| Skip This | Why |
|-----------|-----|
| Full sentences | Too slow — use keywords |
| Articles (a, the) | Not needed for notes |
| Common verbs (is, was, has) | Implied by context |
| Words you already know | Focus on new/specific information |
| Every detail | Focus on main ideas + testable details |

## Abbreviation System

Speed up your writing with consistent abbreviations:

| Full Word | Abbreviation |
|-----------|-------------|
| because | b/c |
| with | w/ |
| without | w/o |
| different / difference | diff |
| important | imp |
| example | ex |

## Note-Taking During the Audio

### Before the Audio Starts (30 seconds)

1. **Read the questions** — know what you need to listen for
2. **Predict the topic** — from the question context
3. **Prepare your paper** — draw a quick grid or list

### While Listening

| If You Hear... | Write... |
|----------------|----------|
| "The main point is..." | ⭐ [that point] |
| "There are three reasons..." | 1. __ 2. __ 3. __ |
| "However / But / On the other hand" | ≠ [the contrast] |
| A number or date | The exact number |
| A name | The exact spelling (if spoken slowly) |
| "For example / Such as" | ex: [brief example] |
| "In conclusion / To summarize" | ⭐ [summary] |

### After the Audio (30 seconds)

1. **Review your notes** — fill in any gaps from memory
2. **Check your answers** — make sure they're complete
3. **Move on** — don't dwell on missed questions

## What If You Miss Something?

| Situation | Action |
|-----------|--------|
| Missed a word | Don't panic — keep listening for the next point |
| Missed a whole sentence | Skip it — answer what you can and move on |
| Missed a whole section | Use context from surrounding questions to guess |
| Don't know the answer | Make an educated guess and move on |

> **Never leave a blank answer.** There's no penalty for wrong answers in IELTS or TOEFL.

---

## ⚠️ Thai Speaker Traps

| Trap | Problem | Fix |
|------|---------|-----|
| Trying to write everything | Too slow, miss the next point | Write keywords only |
| Writing in Thai | Mixing languages slows you down | Practice writing notes in English |
| Panicking when you miss a word | You miss the next 3 sentences while panicking | Accept it and keep listening |
| Not reading questions first | You don't know what to listen for | Always preview questions before audio starts |
| Translating in your head | By the time you translate, the audio moved on | Practice listening in English without translating |
