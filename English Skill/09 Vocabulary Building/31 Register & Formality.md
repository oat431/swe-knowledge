# Register & Formality

Register is the level of formality in your language. IELTS and TOEFL expect **academic register** in writing and speaking — not casual conversation English. Using "kids" in an essay is like wearing shorts to a job interview.

---

## The Register Scale

| Level | Context | Example |
|-------|---------|---------|
| **Very Formal** | Academic papers, legal documents | "The findings indicate a significant correlation." |
| **Formal** | Essays, reports, presentations | "The results show a strong connection." |
| **Neutral** | Standard written English | "The results show a strong link." |
| **Informal** | Conversation, texts, emails to friends | "The results show they're really connected." |
| **Very Informal** | Slang, close friends | "Yo, turns out they're super linked." |

> **IELTS Writing & Speaking:** Aim for **Formal** to **Neutral**.
> **TOEFL Writing & Speaking:** Aim for **Formal** to **Neutral**.

## Informal → Formal Replacements

### Words

| ❌ Informal | ✅ Formal / Academic | Context |
|-----------|---------------------|---------|
| kids | **children** | essays, reports |
| a lot of | **numerous / considerable / substantial** | academic writing |
| big | **significant / substantial / major** | describing importance |
| small | **minor / minimal / negligible** | describing scale |
| good | **beneficial / effective / positive** | evaluating outcomes |
| bad | **detrimental / adverse / negative** | evaluating outcomes |
| get | **obtain / acquire / receive** | formal contexts |
| give | **provide / offer / deliver** | formal contexts |
| show | **demonstrate / illustrate / indicate** | academic writing |
| help | **assist / facilitate / contribute to** | formal contexts |
| think | **believe / consider / maintain** | expressing opinions |
| want | **desire / wish / aim to** | formal requests |
| need | **require / necessitate** | formal writing |
| find out | **discover / determine / establish** | research contexts |
| look at | **examine / investigate / analyse** | academic writing |
| go up | **increase / rise / grow** | describing trends |
| go down | **decrease / decline / diminish** | describing trends |
| thing | **factor / element / aspect / component** | academic writing |
| stuff | **material / content / items** | formal writing |
| OK / okay | **acceptable / adequate / satisfactory** | formal contexts |
| really | **significantly / considerably / notably** | academic writing |
| very | **extremely / highly / substantially** | academic writing |
| totally | **completely / entirely / absolutely** | formal writing |
| pretty (adverb) | **rather / fairly / relatively** | formal writing |
| gonna | **going to** | always in writing |
| wanna | **want to** | always in writing |
| gotta | **have to / must** | always in writing |

### Phrases

| ❌ Informal | ✅ Formal / Academic |
|-----------|---------------------|
| In a nutshell | **In summary / To summarize** |
| At the end of the day | **Ultimately / Ultimately** |
| More and more | **Increasingly / A growing number of** |
| Nowadays | **In contemporary society / Currently** |
| As a matter of fact | **In fact / Indeed** |
| On the other hand | **Conversely / In contrast** (already formal ✅) |
| Like (filler) | Remove entirely |
| You know (filler) | Remove entirely |
| Kind of / Sort of | **Somewhat / To some extent** |

### Contractions

| ❌ Informal | ✅ Formal |
|-----------|----------|
| don't | **do not** |
| doesn't | **does not** |
| can't | **cannot** |
| won't | **will not** |
| it's | **it is** (or rephrase) |
| they're | **they are** |
| there's | **there is** |
| I've | **I have** |
| wouldn't | **would not** |

> **Rule:** In IELTS/TOEFL Writing, **avoid all contractions**. In Speaking, contractions are natural and expected.

## Register in IELTS Task 2 / TOEFL Independent Writing

### ❌ Too Informal
> "I think social media is really bad for kids. They spend a lot of time on their phones and it's totally messing up their sleep."

### ✅ Appropriate Academic Register
> "I believe social media has a detrimental effect on children. They spend considerable time on their devices, which significantly disrupts their sleep patterns."

### Changes Made:
| Informal | Formal |
|----------|--------|
| I think | I believe |
| really bad | detrimental |
| kids | children |
| a lot of time | considerable time |
| phones | devices |
| it's totally messing up | significantly disrupts |
| their sleep | their sleep patterns |

## Register in IELTS Speaking

Speaking is different from writing. **Don't sound like a textbook.** Aim for clear, natural, semi-formal English.

| ❌ Too Casual | ❌ Too Formal | ✅ Just Right |
|--------------|-------------|---------------|
| "Yeah, it's like, super cool I guess." | "I find the phenomenon to be quite fascinating." | "Yes, I find it quite interesting." |
| "Nah, that's dumb." | "I do not concur with that perspective." | "I don't agree with that view." |
| "Dunno, never really thought about it." | "I have not previously contemplated that matter." | "I haven't really thought about it before, but..." |

## First Person in Academic Writing

| Acceptable? | IELTS | TOEFL |
|-------------|-------|-------|
| "I think..." | ✅ OK for opinion essays | ✅ Expected |
| "In my opinion..." | ✅ OK | ✅ OK |
| "This essay will discuss..." | ⚠️ Avoid (impersonal is better) | ❌ Avoid |
| "One might argue..." | ✅ More formal | ✅ Good |
| "It is evident that..." | ✅ Best for formal essays | ✅ Best |

## ⚠️ Thai Speaker Traps

| Trap | Example | Fix |
|------|---------|-----|
| Using "nowadays" too much | "Nowadays, people use phones. Nowadays, technology is advanced." | Use: "Currently," "In recent years," "Today" |
| Overusing "a lot of" | "There are a lot of benefits." | "There are **numerous / significant** benefits." |
| Using "thing" for everything | "Technology is a good thing." | "Technology is a **beneficial development**." |
| Mixing registers | "The government should fix this thing." | "The government should **address** this **issue**." |
| Using contractions in essays | "It's important that we don't..." | "It is important that we do not..." |
