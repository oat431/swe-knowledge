## 6. Icon Usage Guidelines

| Rule | Example |
|------|---------|
| [Always pair with text for important actions] | ✅ [🔍 Search] ❌ [🔍 alone] |
| [Use consistent size within context] | ✅ [All nav icons 20px] ❌ [Mixed sizes] |
| [Use semantic colors for status] | ✅ [Green check for success] ❌ [Blue check for success] |
| [Maintain clear space around icons] | ✅ [8px padding] ❌ [Icon touching text] |
| [Don't use decorative icons for actions] | ✅ [Clear action icon] ❌ [Ambiguous icon] |
